# Contributing to cBTC

Thank you for your interest in the cBTC protocol.

## How to contribute

- **Issues**: Use GitHub Issues to report bugs, ask questions, or propose changes.
- **Pull Requests**: Fork the repo, create a branch, and open a PR with a clear description.

## Development

- This project currently targets **Bitcoin regtest** only.
- You need:
  - Bitcoin Core
  - Python 3.10+
  - `python-bitcoinrpc` library

## Code Style

- Prefer clear, explicit code over clever tricks.
- Include comments for all protocol-related logic.
